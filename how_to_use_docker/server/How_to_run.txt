1. Скомпилируй fcgi_config.c - 
``````````````````````````````````````````````````````
gcc fcgi_config.c -lfcgi -o fcgi_config
``````````````````````````````````````````````````````

2. Запусти FastCGI - 
``````````````````````````````````````````````````````
spawn-fcgi -a 127.0.0.1 -p 8080 -f ./fcgi_config -F 1
``````````````````````````````````````````````````````
