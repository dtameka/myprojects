3DViewer v1.0

Программа позволяет загружать каркасную модель из файла формата obj (поддержка только списка вершин и поверхностей)

Движение модели происходит посредством заполнением нужного поля ввода и нажатием определённой кнопки +/-

Поворот модели происходит посредством заполнением нужного поля ввода и нажатием определённой кнопки +/-

Масштаб модели происходит посредством перетягивания ползунка в левую часть(для уменьшения) или в правую часть(для увеличения)

По нажатию правой кнопкой мыши по виджету экрана отрисовки, вы можете выбрать одно из предоставленых действий.

Также вы можете вызвать это меню путём нажатия лкм на слово "файл" в левом верхнем углу.

При переходе в настройки вам доступен выбор

  1. Цвет фона
  2. Цвет линий
  3. Цвет вершин
  4. Тип проекции
  5. Тип линий
  6. Тип вершин
  7. Толщина линий
  8. Размер вершин

Текущие настройки сохраняются при закрытии программы.

Программмма позволяет сохранять полученные изображения в файл в форматах bmp и jpeg/png.

Программа позволяет записывать небольшие "скринкасты" - текущие пользовательские аффинные преобразования загруженного объекта в gif-анимацию (640x480, 10fps, 5s).
